CLAS12 Monte-Carlo Job Submission Portal

                                                                             Logged in as

Types

Type 1 and 2 corresponds to conditions of CLAS12 experiments that already ran.
Type 3 and 4 can be used for custom user conditions, for example to test a new detector design or a different magnet configurations.


Output

The output is saved on 

	/volatile/clas12/osg/"username"/job_ID

where "username" is your jlab account name and ID is the submission ID


Priority

There is no user priority. Jobs are submitted first come first serve.


Versions

The software versions are:

  • gemc: 4.3.1
  • Coatjava: 6b.3.2

    • We plan to support this and future versions.
